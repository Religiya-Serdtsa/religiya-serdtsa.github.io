# 블로그 운영 공지

이 카테고리는 **Religiya Serdtsa** 블로그 운영과 관련된 공지 사항을 모아 둡니다.

## 공지 유형

- **플랫폼 변경**: 제너레이터 업그레이드, 빌드 방식 변경, 의존성 갱신
- **구조 개편**: 카테고리 추가·삭제, URL 변경, 포스트 이동
- **정책 안내**: 기여 방법, 포스트 작성 규칙, 태그 사용 기준

## 현재 플랫폼

이 블로그는 `bin/bloggen` — cwist sstring + md4c 기반 C 정적 제너레이터로 구동됩니다. GitHub Actions가 `posts/` 디렉터리와 `categories.cfg`를 읽어 `docs/`에 HTML을 생성하고, GitHub Pages가 서빙합니다.

변경 사항이 생기면 이 카테고리에 공지합니다.
